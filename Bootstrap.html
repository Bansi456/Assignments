(2)What is a Bootstrap Container, and how does it work?

Ans-
Containers are a fundamental building block of Bootstrap that contain, pad, and align your content within a given device or viewport.
They work most basic layout element in bootstrap and required when using our defult grid system. While containers can be nested, most layouts do not require a nested container.

Bootstrap come with three different containers:
(1).container,which sets a max-width at each responsive breakpoint
(2).container-fluid,which width-100% at all breakpoint
(3).container-{breskpoint},which is width-100% untill the specified breakpoint

(4)What do you know about the Bootstrap Grid System?

Ans-
Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive. Below is an example and an in-depth look at how the grid comes together.

Grid Column:

Bootstrap's grid system allows up to 12 columns across the page.
If you do not want to use all 12 column individually, you can group the columns together to create wider columns:

Grid Classes:

The bootstrap grid system has four classes:
(1)xs (for phones - screens less than 768px wide)
(2)sm (for tablets - screens equal to or greater than 768px wide)
(3)md (for small laptops - screens equal to or greater than 992px wide)
(4)lg (for laptops and desktops - screens equal to or greater than 1200px wide)

Grid Option

Bootstrap’s grid system can adapt across all six default breakpoints, and any breakpoints you customize. The six default grid tiers are as follows:
(1)Extra small (xs)
(2)Small (sm)
(3)Medium (md)
(4)Large (lg)
(5)Extra large (xl)
(6)Extra extra large (xxl)


(5)What is the difference between Bootstrap 4 and Bootstrap 5

Ans-

Bootstrap 4

(1) It has 5 tier (xs, sm, md, lg, xl).


(2) It has limited colors.




(3) Bootstrap 4 supports both IE 10 and 11.


(4) We cannot modify utilities in bootstrap 4



(5) Columns can be positioned relative


(6) Bootstrap 4 doesn’t have its own SVG icons, we have to use font-awesome for icons.



(7) The card deck is used to create a set of cards with equal width and height.


(8) We have inline-block property and we will get white dropdown as default for dropdown-menu-dark class.


(9) this makes easier to implement vertical designs, and the columns and rows can easily be implemented. the classes justify-center-content can directly be used to align according to the requirement.


(10)It does not support Offcanvas Component.

Bootstrap 5

(1) It has 6 tier (xs, sm, md, lg, xl, xxl).


(2) Extra colors added with the looks, A card improved color palette. there are various shades available to choose.


(3) Bootstrap 5 doesn’t support IE 10 and 11.


(4) Bootstrap 5 gave freedom to modify and also create our own utility.


(5) Columns cannot be positioned relative


(6) Bootstrap 5 have its own SVG icons.




(7) Card deck class in removed in bootstrap



(8) Inline-block property is removed and we will get black dropdown as default for dropdown-menu-dark class.


(9) advanced grid system is made available , also columns don’t have relative positions.





(10) It supports Offcanvas Component(that is it is available now).
